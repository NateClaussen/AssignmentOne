UI -VS- UX

What is the UI?

UI stands for "User Interface". It includes visual elements such as the pages, buttons, pictures, etc. that people see. It is not concerned about how it flows with the other elements on the page. It also does not care if there are any interactive elements, just that everything looks pretty!

What is the UX?

UX stands for "User Experience" with the product. This includes how different components interact, and how that feels to the end user. The UX DOES care about how things flow, and how user friendly your application is
